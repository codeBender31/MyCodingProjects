------------------------------------------------------------------------
Ticket Reservation System
------------------------------------------------------------------------

PURPOSE OF PROJECT: To allow users to safely choose seats from an array of movie theater options. If the desired seats are not available then
it should return the best possible seats available, otherwise advice the user to pick another seat. 
Once the application is done running it will write on to a file a summary of all the tickets that were sold and total profits. 
VERSION or DATE: 7/7/2023
HOW TO START THIS PROJECT: To start this project compile and run the main function and use the A*.txt files as inputs. Then simply follow 
instructions to choose your theater seats.
